# Contributing

Contributions are welcome.

1. Create a focused branch.
2. Preserve Windows PowerShell 5.1 compatibility.
3. Add or update Pester tests for behavioral changes.
4. Run `Invoke-Pester .\tests\CodexMonitor.Tests.ps1`.
5. Open a pull request describing behavior, verification, and privacy impact.

Do not add telemetry or parse additional session fields without a documented need and explicit privacy review.
