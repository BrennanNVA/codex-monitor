# Codex Monitor v0.2.1

Codex Monitor v0.2.1 is a focused patch release that removes visible blinking during live dashboard updates.

## Fix

- Build each dashboard refresh in memory and emit it as one console write.
- Replace rows before clearing unused row tails and leftover screen space, eliminating the blank interval between updates.
- Preserve the existing colors, keyboard controls, event-driven refresh rate, and token-accounting behavior.

## Verification

- Add a regression test that proves frame content precedes terminal cleanup commands.
- Test the renderer under Windows PowerShell 5.1 and PowerShell 7.

## Upgrade

Download and extract `codex-monitor-v0.2.1.zip`, then run `Start-Codex-Monitor.cmd`. No configuration or persistent-data migration is required.

## Compatibility and rollback

Codex Monitor supports Windows PowerShell 5.1 and PowerShell 7. To roll back, replace the v0.2.1 files with v0.2.0; no stored data requires migration.
